## Getting Started

### Logging In

1. Navigate to the application URL
2. Enter your credentials:
   - **Username**: admin
   - **Password**: admin123
3. Click "Login"

### Changing Language

The system supports 5 languages: English, Arabic, Dutch, French, and Polish.

To change the language:
1. Click the language selector button in the top-right corner (shows current language with flag)
2. Select your preferred language from the dropdown
3. The interface will update immediately
4. Your selection is saved automatically

**Note**: Arabic language enables right-to-left (RTL) layout automatically.

## Employee Management

### Viewing Employees

1. Click "Employee Management" in the sidebar
2. Use the "Employees" tab to see the full list
3. **Search**: Type in the search box to find employees by name, ID, department, or position
4. **Filter**: Click the Filter button to apply advanced filters
5. **Pagination**: Use the page controls at the bottom to navigate through employees

### Adding a New Employee

1. Go to Employee Management
2. Click "Add Employee" button
3. Fill in the employee details:
   - Personal Information (Name, Email, Phone)
   - Job Information (Department, Position, Location)
   - Employment Details (Join Date, Employee ID)
   - Salary Information
4. Click "Save" to create the employee

### Viewing Employee Profile

1. In the employee list, click on an employee name or click the "View" action
2. The profile shows:
   - Personal information
   - Employment details
   - Recent attendance
   - Leave balance
   - Performance metrics

### Editing Employee Information

1. View the employee profile
2. Click "Edit" button
3. Update the necessary fields
4. Click "Save Changes"

### Employee Tabs

- **Employees**: Main employee list
- **Attendance**: Quick attendance overview for all employees
- **Payroll**: Payroll processing and history
- **Performance**: Performance reviews and ratings
- **Training**: Training assignments and completion
- **Assets**: Assets assigned to employees
- **Expense**: Expense claims and approvals

## Attendance Tracking

### Today's Attendance

1. Go to Attendance Tracking
2. The "Today" tab shows:
   - Who is present, late, or absent
   - Check-in and check-out times
   - Total work hours
   - Real-time status updates

### Calendar View

1. Click the "Calendar" tab
2. **Select Month**: Use the month dropdown to view different months
3. **View Details**: Click on any day to see attendance details
4. **Statistics**: Monthly totals shown at the bottom
   - Total Present Days
   - Total Absent Days
   - Total Late Arrivals
   - Half Days

### Analytics

1. Click the "Analytics" tab
2. View:
   - Department-wise attendance (bar chart)
   - Attendance distribution (pie chart)
   - Monthly trends (line chart)

### Exporting Attendance Data

1. Select the desired month/period
2. Click "Export" button
3. Choose format (Excel, CSV, PDF)
4. File will be downloaded

## Leave Management

### Applying for Leave

1. Go to Leave Management
2. Click the "My Leaves" tab
3. Click "Apply Leave"
4. Fill in:
   - Leave Type (Annual, Sick, Casual, etc.)
   - Start Date
   - End Date
   - Reason
5. Click "Submit"

### Viewing Leave Balance

1. Go to Leave Management
2. "Leave Balance" section shows:
   - Available leave days by type
   - Used leave days
   - Pending requests
   - Leave history

### Approving Leave Requests (Manager/HR)

1. Go to Leave Management
2. "Leave Requests" tab shows all pending requests
3. Click on a request to view details
4. Click "Approve" or "Reject"
5. Add comments if needed

### Leave Calendar

View all team leaves on a calendar:
1. Click "Calendar" tab
2. See color-coded leave periods
3. Filter by department or employee

## Payroll Management

### Viewing Payroll

1. Go to Payroll Management
2. Overview shows:
   - Total Payroll for current month
   - Processed vs Pending
   - Average Salary
   - Payroll trends

### Processing Payroll

1. Click "Generate Payroll" button
2. Select month and year
3. Review employee list and amounts
4. Verify calculations
5. Click "Process Payroll"
6. Confirm the action

### Generating Payslips

1. Select the payroll period
2. Click "Generate Payslips"
3. Payslips are created for all employees
4. Download individual or bulk payslips

### Payroll Components

Each payslip shows:
- **Basic Salary**: Base pay
- **Allowances**: HRA, Transport, etc.
- **Deductions**: Tax, Insurance, etc.
- **Net Salary**: Take-home pay

3. Fill in job details:

   - Job Description
   - Requirements

   - Interview Completed

## Common Actions

### Search

All listing pages have a search bar:
1. Type your search term
2. Results filter in real-time
3. Search works across multiple fields

### Filtering

Advanced filtering available on most pages:
1. Click "Filter" button
2. Select filter criteria
3. Apply filters
4. Clear filters when done

### Pagination

Navigate through large lists:
- **Items per page**: Usually 10-20 items
- **Navigation**: Use page numbers or Previous/Next
- **Jump to page**: Click specific page number

### Exporting Data

Most modules allow data export:
1. Click "Export" button
2. Choose format:
   - Excel (.xlsx)
   - CSV (.csv)
   - PDF (.pdf)
3. File downloads automatically

### Back Navigation

When viewing details or editing:
1. Click "Back" button or breadcrumb
2. Returns to previous view
3. **Tab State Preserved**: Returns to the same tab you were on

## Tips & Best Practices

### Navigation
- Use breadcrumbs to understand your location
- Sidebar shows active module in highlighted state
- Use browser back button sparingly (use in-app navigation)

### Data Entry
- Required fields are marked with *
- Validation messages appear near fields
- Save frequently when editing long forms

### Performance
- Use filters to reduce large lists
- Export data for offline analysis
- Calendar views load one month at a time

## Troubleshooting

### Can't find an employee?
- Check spelling in search
- Try searching by ID or department
- Clear all filters
- Check if employee is inactive

### Language not changing?
- Clear browser cache
- Check if browser blocks localStorage
- Refresh the page

### Data not loading?
- Check internet connection
- Refresh the page
- Clear browser cache
- Try different browser

### Export not working?
- Disable popup blockers
- Check download folder settings
- Try different export format
